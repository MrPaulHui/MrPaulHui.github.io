# 发表一篇新blog的正确姿势

1. python create_new_post.py 输入标题和标签 在_post文件夹下生成待写的md（暂时有问题，先手动写）

2. 完成blog编写

3. git add .

   git commit -m ''

   git push

4. 等一段时间后，刷新page，即可

# 本地运行调试

项目目录下，执行jekyll server
